## Project Set Up

1. Download and install IntelliJ IDEA
2. Download and install JDK 8
3. Set the $JAVA_HOME path variable
3.a /usr/libexec/java_home -v 1.8
3.b vim ~/.zshrc
4. Install Maven
5. Run mvn install
